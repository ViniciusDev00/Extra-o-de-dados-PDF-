# ORFA PDF Extractor

## Overview
ORFA PDF Extractor is a robust tool designed to extract text and data from PDF files efficiently. It is particularly suited for users who need to automate the process of PDF data extraction and make their workflow smoother.

## Features
- **User-Friendly Interface**: Simple and intuitive interface for easy navigation.
- **Fast Extraction**: High-speed extraction of text and data from PDFs.
- **Support for Multiple PDF Formats**: Compatible with various PDF formats and structures.
- **Export Options**: Extracted data can be exported into multiple formats for easier data manipulation.
- **GitHub Pages Demo**: [Live Demo](https://viniciusdev00.github.io/Extrator-de-PDF/)

## Usage Instructions
1. **Installation**: Clone the repository:
   ```bash
   git clone https://github.com/ViniciusDev00/Extrator-de-PDF.git
   ```
2. Go into the project directory:
   ```bash
   cd Extrator-de-PDF
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```
5. Follow the instructions in the interface to select your PDF and extract the required data.
